## рЯПЧ Local Development

### Prerequisites

- **Node.js** вЙ• 20
- **npm** вЙ• 9 (or your package manager of choice)

### Setup

```bash
# Clone the repository
git clone git@github.com:mataza060503/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash

npm start
```

| `SMTP_PASS` | SMTP password |
| `CONTACT_EMAIL` | Recipient email address for the contact form |
